# Contributor Covenant Code of Conduct (Short)

Our project follows the Contributor Covenant v2.1. We expect everyone participating in the project to abide by the code of conduct.

Be Respectful

- Be friendly and welcoming. Treat others with respect.
- Assume good intent and be open to feedback.

Unacceptable behavior

- Harassment, abusive language, or personal attacks are not tolerated.
- Discriminatory comments or hateful content are unacceptable.

Reporting

If you experience or witness unacceptable behavior, please open an issue addressed to the maintainers or email the project owner privately at the address in `README.md`.

Enforcement

Project maintainers may remove or modify content or ban users who violate the code of conduct.

For the full text see: https://www.contributor-covenant.org/version/2/1/code_of_conduct/
